## SwitchStatement Grammar: `SwitchStatement: switch [VariableRef] (case (Expression,) ":" Body)* [default Body]` Similar to js switch, but: 1. no fall-through 2. each 'case' can have several expressions, comma separated, then ':' 3. if no var specified, select first true expression Examples: var a=2 b=3 c=5 switch a case 2,4,6: print 'even' case 1,3,5: print 'odd' default: print 'idk' no var, select first true expression switch case a is 3 or b < 10: print 'option 1' case b >= 10, a<0, c is 5: print 'option 2' default: print 'other'
